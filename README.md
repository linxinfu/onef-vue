# 简介

Onef(one for all)是一个个人私密信息管理方案，基于 Vue 最新技术栈开发，内置了 i18 国际化解决方案，动态路由和权限验证。

## 技术栈

前端技术栈： vue2 + vuex + vue-router + axios + ES6 + sass + element-ui

## 后端仓库

[onef-kotlin](https://github.com/linxinfu/onef-kotlin)

## 使用

```
# 安装依赖
$ yarn install

# 启动
$ yarn start
```
